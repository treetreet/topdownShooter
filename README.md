## 플레이 방법
2팀으로 나누어 즐길 수 있는 슈팅 게임입니다.

게임 로비를 직접 만들어서 다른 사람이 참가할 수 있습니다.

WASD로 이동, 마우스 좌클릭으로 총알을 발사하고 R을 눌러 재장전을 할 수 있습니다.

플레이어는 총알을 맞고 사망한 후 일정 시간 후에 각자 정해진 곳에서 부활합니다.

점령지점은 플레이어가 직접 올라가서 점령할 수 있습니다. 

점령지점에 올라간 팀의 게이지가 채워지며, 모두 채워지면 그때부터는 점수가 오르기 시작합니다.

점령지점에 두 팀이 모두 올라가 있으면 게이지가 채워지지 않습니다.

한 팀의 점수가 100에 도달하면 승리합니다.
